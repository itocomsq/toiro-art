toiro art. 編集ツール

このページはお店のサイトを編集するための管理者用ツールです（お客様は使いません）。 編集後、右上の「フォルダを選んで保存」から サイトのフォルダ（このeditor.htmlがある場所） を選ぶと、 画像や商品情報のファイルを直接書き換えます。保存後は忘れずに git add → commit → push してください（手順は README.md 参照）。

※ Chrome または Edge で、http://localhost... のようにサーバー経由で開いている必要があります（詳しくはREADME参照）。
